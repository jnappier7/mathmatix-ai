# Avatar Images Needed

This document lists all the creature avatar images that need to be created for the student avatar selection system.

## Directory
All images should be placed in: `/public/images/avatars/`

## Image Specifications
- **Format**: PNG with transparent background
- **Size**: 200x200px minimum (will be displayed at 100x100px in circles)
- **Style**: Fun, colorful, kid-friendly illustrations (Blooket-style)
- **Consistency**: All avatars should have similar art style and quality

## Design Notes

### Rarity Visual Cues
- **Common**: Simple, solid colors
- **Uncommon**: Add 1-2 accent colors or simple effects
- **Rare**: Add gradients or sparkles
- **Epic**: Add glow effects or special elements
- **Legendary**: Add animated-ready effects (shimmer, gold accents)

### Character Traits to Emphasize
- **Lion**: Confident, leader-like
- **Dragon**: Powerful but friendly
- **Panda**: Chill, approachable
- **Owl**: Intelligent, studious
- **Fox**: Quick, clever
- **Tiger**: Focused, determined
- **Unicorn**: Magical, special
- **Penguin**: Cool under pressure
- **Wolf**: Team player, persistent
- **Phoenix**: Resilient, rising from challenges
- **Shark**: Precise, efficient
- **Eagle**: Big picture thinker
- **Octopus**: Multi-tasker
- **Raccoon**: Creative problem solver
- **Robot**: Logical, calculated
- **Alien**: Out-of-the-box thinker
- **Ninja**: Strategic, stealthy
- **Wizard**: Master of skills
- **Dinosaur**: Prehistoric power
